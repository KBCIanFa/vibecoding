The philosophy is dead simple: make something real, make it free, and make it actually work. No npm install. No webpack config. No deployment pipeline. Just a file and a URL.

Everything here was built during an intensive vibe-coding run — a solo session of describing what I wanted, letting Claude write the bones, then iterating until it felt right.

  • 📄
    Single-file HTML — the whole app lives in one file. Open it, it works.
  • 🚫
    Zero build steps — no npm, no Webpack, no "just run this script first."
  • 🇦🇺
    Australia-first — specs, roads, humour, and prices are all AU by default.
  • 🤖
    AI as co-pilot — Claude writes the code; I steer, test, and ship.
  • 🆓
    Free forever — hosted on GitHub Pages. No paywalls, no accounts needed.
