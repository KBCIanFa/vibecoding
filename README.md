# Vibe Coding

A small GitHub Pages hub for Ian Fraser's vibe-coded web projects, tutorials, and experiments.

This repo is built around a simple idea:

> Build it. Ship it. Use it.

No build pipeline. No backend. No npm install. Just plain HTML, CSS, JavaScript, GitHub Pages, and a willingness to keep iterating until the thing actually works.

## Live Site

Visit the live site here:

[https://kbcianfa.github.io/vibecoding/](https://kbcianfa.github.io/vibecoding/)

## What This Repo Is

This repo is a public landing page and tutorial set for practical vibe coding.

It includes:

- a project showcase page
- a beginner-friendly vibe coding tutorial
- a deeper explanation of why vibe coding matters
- links to live tools built with the same approach
- single-file HTML pages that can be opened, edited, forked, and hosted easily
